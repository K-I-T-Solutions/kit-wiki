> **Hinweis:** „Auth“ = Authentifizierung (du bist wer du sagst), „AuthZ“ = Autorisierung (du darfst was du willst).

### Netzwerk & DNS
- **FQDN** – Fully Qualified Domain Name (z. B. `wiki.kit-it-koblenz.de`)
- **A/AAAA** – IPv4/IPv6-Hosteintrag
- **CNAME** – Alias auf anderen Host
- **MX** – Mail-Exchanger (E-Mail-Zustellung)
- **PTR** – Reverse DNS (IP → Name)
- **SRV** – Service-Eintrag (z. B. `_sip._tcp`)
- **TTL** – Time To Live (Caching-Zeit in DNS)
- **DHCP** – Dynamic Host Config Protocol (IP-Vergabe)
- **NAT** – Network Address Translation
- **CIDR** – Netznotation (z. B. `192.168.10.0/24`)
- **VLAN** – Virtuelles LAN (Netzsegmentierung)

### Mail
- **IMAP/SMTP** – Empfang/Versand von E-Mails
- **SPF/DKIM/DMARC** – Absender-Authentizität/Integrität/Policy
- **MTA/MDA/MUA** – Transport/Delivery/User-Agent (Server/Klient)

### Identity & Security
- **SSO** – Single Sign-On
- **2FA/MFA** – Zwei-/Mehrfaktor-Auth
- **TOTP** – App-basierter Einmalcode (z. B. Aegis/Authenticator)
- **SAML / OIDC(OAuth2)** – Föderierte Anmeldung/Token-Flows  
- **IdP / SP** – Identity Provider / Service Provider
- **RBAC/ABAC** – Role-/Attribute-based Access Control
- **ZT / ZTNA / SASE** – Zero Trust / Network Access / Secure Access Service Edge
- **EDR/MDM/RMM** – Endpoint Detection & Response / Mobile Device Mgmt / Remote Monitoring & Mgmt
- **SOC / SIEM** – Security Operations Center / Security Info & Event Mgmt
- **RTO/RPO** – Recovery Time/Point Objective

### Betrieb & Prozesse
- **SOP** – Standard Operating Procedure
- **RCA** – Root Cause Analysis
- **KB / KBA** – Knowledge Base / Artikel
- **SLA** – Service Level Agreement
- **PoC** – Proof of Concept
- **L1/L2/L3** – Supportlevel
- **LTS / EOL/EOS** – Long-Term-Support / End of Life/Service

### Dev & Ops (nerdy)
- **CI/CD** – Continuous Integration/Delivery
- **IaC** – Infrastructure as Code (z. B. Ansible/Terraform)
- **YAML/TOML/JSON** – Config-Formate
- **RFC** – Request for Comments (Standards)
- **ETA/TBD/WIP** – Ankunftszeit / wird definiert / in Arbeit
